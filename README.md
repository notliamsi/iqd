# IQD
Install Qemu on Debian (IQD) is a simple automation script designed to expedite and simplify the installation process of QEMU on Debian-based systems.

## How to use it
1. Clone this repo
```bash
git clone https://github.com/notliamsi/iqd
```
2. Run the script
```bash
./iqd
```
3. Reboot
```bash
systemctl reboot 
```
